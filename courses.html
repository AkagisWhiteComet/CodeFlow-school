Frontend-раработчик

Создаёт клиентскую часть сайта (дизайн). В последнее время спрос на эту специальность растёт.
Советуем выбирать эту профессию, если вы хотите заниматься только дизайном.

На курсе вы научитесь:

  • работать с HTML, CSS
  • разрабатывать дизайн для веб-приложений и сайтов
  • разбираться в JavaScript и TypeScript
  • пользоваться методами ООП и библиотекой React
  • взаимодействовать с командой посредством Git
  • публиковать веб-приложения и сайты в сети
  • тестировать и править готовый продукт
  • мыслить как опытный frontend-разработчик

График занятий: по субботам и воскресеньям в 15:00 по МСК, за отдельную плату индивидуально в любое удобное для Вас время.
Срок обучения: 10 месяцев.

СТОИМОСТЬ: 4500₽/мес.

Backend-раработчик

Занимается функциональной частью веб-сайта: создаёт серверную часть, настраивает базу данных и разрабатывает API для взаимодействия с фронтендом. Советуем выбирать, если вам не нравится разрабатывать дизайн.

Вы научитесь:

  • писать и читать код на Python
  • писать автотесты и UI-тесты
  • пользоваться фреймворками (Django, Flask)
  • отправлять HTTP-запросы
  • работать с базами данных SQL, SQLAlchemy, PostgreSQL
  • работать с ПО для развертывания и запуска веб-приложений, сайтов
  • отслеживать ошибки кода и править его
  • применять паттерны проектирования

График занятий: по субботам и воскресеньям в 15:00 по МСК, за отдельную плату индивидуально в любое удобное для Вас время.
Срок обучения: 1 год.

СТОИМОСТЬ: 7000₽/мес.
